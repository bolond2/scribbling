# Perspectives — notes to self

For posts on economics, politics, current events, or Christian faith — topics I'm genuinely
interested in but not "expert" on, and areas I might not always want front-and-center if
someone stumbles onto this site looking for a professional writing sample.

## Default mode: explain, don't argue

Default to explainer mode over opinion mode: "here's what's actually happening and why it
matters" rather than "here's what I think you should believe." This is harder and more
valuable than it sounds, and it's the more useful skill to practice anyway — representing a
position accurately, including ones I don't hold, is close to writing for stakeholders who
don't share my view.

Opinion posts aren't banned — just be aware that's a different, lower-guardrail mode, and use
it deliberately, not by default.

## Reminders
- These posts are for me, not for recruiting an audience or winning an argument.
- Not planning to point employers here by default — but keep in mind someone might find it,
  so write like a person you respect could read it and disagree without it feeling like an
  ambush.
- If a post ever feels more like venting than explaining, let it sit a day before publishing.
